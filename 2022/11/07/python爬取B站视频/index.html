python爬取B站视频

Posted on 2022-11-07

一、前言

今天逛B站时，发现了一个有趣的 UP 主，叫“手工耿”，就想把他的视频全爬下来。

首先看下手工耿的个人空间。

地址：https://space.bilibili.com/280793434

点击更多，打开他的全部视频。

查看请求，不难发现，视频链接是通过一个接口获得的：

 

 这个 API 接口是这样的：

https://api.bilibili.com/x/space/arc/search?mid=280793434&ps=40&tid=0&pn=1

mid 就是用户的一种 id，可以通过用户首页链接后面的数字获取这个 mid。

                                                 mid：↑↑↑↑↑↑↑↑↑

ps 是说1页多少个视频。

tid 一直为 0 即可。

pn 是翻页。

知道这些，就可以直接写代码，获取所有视频的链接了。

  1. import requests
  2. import json
  3. import math
  4. import re
  5. import os
  6.  
  7. os.makedirs('视频',exist_ok = True)
  8. res = requests.get('https://api.bilibili.com/x/space/arc/search?mid=280793434&tid=0&pn=1&ps=40')
  9. info = json.loads(res.text)
  10. video_count = info['data']['page']['count']
  11. print(video_count)
  12. pages = math.ceil(video_count/40)  #向上取整获取页数
  13. print(pages)
  14. videos_list = []
  15. for pn in range(1,pages+1):
  16.     res = requests.get('https://api.bilibili.com/x/space/arc/search?mid=280793434&tid=0&pn=%s&ps=40'%(str(pn)))
  17.     info = json.loads(res.text)
  18.     vlist = info['data']['list']['vlist']
  19.     for video in vlist:
  20.         title = video['title']
  21.         bvid = video['bvid']
  22.         vurl = 'https://www.bilibili.com/video/' + bvid
  23.         videos_list.append([title, vurl])
  24. print('共 %d 个视频' % len(videos_list))
  25. for video in videos_list:
  26.     print(video[0] + ':' + video[1])

二、视频下载

有了链接，接下来就可以下载视频了。

打开视频页面，抓包分析。

URL：https://www.bilibili.com/video/BV1Ya4y1a7DC

这个是 PC 端的链接，打开这个视频，你可能花费九牛二虎之力，也找不到视频的URL。

我们可以开启移动端模式。这回再刷新页面，我们就能轻松找到视频的下载地址了。

这是其中一个视频下载链接： 

https://cn-gddg-dx-bcache-24.bilivideo.com/upgcxcode/20/80/315408020/315408020-1-16.mp4?e=ig8euxZM2rNcNbdlhoNvNC8BqJIzNbfq9rVEuxTEnE8L5F6VnEsSTx0vkX8fqJeYTj_lta53NCM=&uipk=5&nbs=1&deadline=1627629750&gen=playurlv2&os=bcache&oi=243224835&trid=000045e5af001a6b4764a3dcef15ec634992h&platform=html5&upsig=ccd54e4e7a36d3196a4878e3d9a0a211&uparams=e,uipk,nbs,deadline,gen,os,oi,trid,platform&cdnid=61324&mid=0&bvc=vod&nettype=0&logo=80000000 

下载成功！

 现在我们可以编写获取视频下载URL的程序了：

  1. url_list = []
  2. header = {
  3.     'user-agent': 'Mozilla/5.0 (Linux; Android 6.0; Nexus 5 Build/MRA58N) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/92.0.4515.107 Mobile Safari/537.36 Edg/92.0.902.55'
  4.     }
  5. for i in videos_list:
  6.     res = requests.get(i[1],headers=header)
  7.     res.encoding = 'utf-8'
  8.     url = re.findall('https.*80000000',str(res.text))[0]
  9.     url_list.append([i[0],url])
  10.     print('正在获取"%s"的url'%(i[0]))

接下来下载视频：

  1. for i in url_list:
  2.     try:
  3.         with open('./视频/'+i[0]+'.mp4','wb') as f:
  4.             print('正在下载"%s"'%(i[0]+'.mp4'))
  5.             f.write(requests.get(i[1],headers=header).content)
  6.     except:
  7.         print('下载"%s"失败'%(i[0]))
